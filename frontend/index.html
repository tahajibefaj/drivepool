One-time Setup

Deploy in 15 minutes

Free hosting on Render.com. Your drives sleep when not in use and wake up in seconds when you visit.

1

Fork the GitHub repo

Fork DrivePool to your GitHub account (one click).

2

Get Google credentials (free)

Create a project at console.cloud.google.com → Enable Drive API → Create OAuth credentials → Download client_secrets.json

3

Deploy on Render.com

New Web Service → connect your GitHub repo → it auto-detects render.yaml → deploy.

4

Set environment variables

Paste your client_secrets.json content as the GOOGLE_SECRETS_JSON env var. Set BASE_URL to your Render URL.

5

Connect your Google accounts

Visit your Render URL → Open Dashboard → Add Account → repeat for each Google account.

Security

Your files stay in Google Drive

DrivePool never stores, copies, or caches your actual files. It only manages access and routes operations.

🔐

OAuth 2.0

You sign in directly through Google. DrivePool never sees your password or credentials.

📦

No file storage

Files are streamed directly to/from Google Drive. Nothing is stored on the hosting server.

🔑

Encrypted tokens

OAuth refresh tokens are stored only on your persistent disk. Not shared with anyone.

🚫

Revoke anytime

Remove any account from DrivePool or revoke access directly in your Google account settings.
